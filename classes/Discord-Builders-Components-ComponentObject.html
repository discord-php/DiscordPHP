ComponentObject extends Component
in package
  • Application

AbstractYes
ComponentObject.php : 27

Components allow you to style and structure your messages, modals, and interactions.

They are interactive elements that can create rich user experiences in your Discord applications.

Components are a field on the message object and modal. You can use them when creating messages or responding to an interaction, like an application command.
